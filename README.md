# Ni Aasre Da Aasra NGO - Web Development Project

Welcome to the repository containing the raw source code and assets for the web pages developed for **Ni Aasre Da Aasra**, a non-governmental organization. 

## Project Overview
This project involved designing and developing specific informational web pages for the Ni Aasre Da Aasra NGO to help them effectively showcase their community recognition and media presence. The raw code housed in this repository was successfully integrated and deployed onto their official website.

## Live Pages
The finalized pages are currently live and can be viewed here:
* **Awards Page:** [Seva Karo Awards](https://sevakaro.in/index.php?route=information/awards)
* **Media Clippings Page:** [Seva Karo Clippings](https://sevakaro.in/index.php?route=information/clippings)
